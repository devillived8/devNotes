# devNotes — шпаргалка для front-end разработчика 🚀

**devNotes** — это небольшой справочник по основам веб-разработки.  
Сайт создан в процессе обучения на Яндекс Практикуме, чтобы структурировать знания и иметь под рукой быстрый доступ к базовой информации по HTML, CSS и не только.

👉 **[Смотреть демо](https://devillived8.github.io/devNotes/)**

---

## ✨ О проекте

- 📚 **HTML** — структура документа, семантические теги, формы, атрибуты
- 🎨 **CSS** — базовая типографика, flex, grid, адаптив (в разработке)
- ⚙️ **JavaScript** — интерактив (бургер-меню, якоря)
- 🧠 **Доступность** — aria-атрибуты, семантика, навигация с клавиатуры

Проект постоянно дополняется по мере прохождения курса.

---

## 🛠 Технологии

- **HTML5** — семантическая вёрстка
- **CSS3** — переменные, flex, grid, адаптив
- **JavaScript (ES6+)** — классы, модули, события
- **БЭМ** — методология именования классов
- **Git / GitHub Pages** — хостинг и контроль версий
